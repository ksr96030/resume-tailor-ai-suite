# AI Resume Tailoring Suite (Hybrid Mode)
## Quick Start:
1. Run setup script:
\`\`\`
bash full_setup.sh
\`\`\`
2. Start system:
\`\`\`
make run
\`\`\`
Access:
- Frontend: http://localhost:3000
- Backend: http://localhost:8080/swagger-ui.html
- AI Service: http://localhost:8081
